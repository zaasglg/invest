# 🤖 Как настроить AI Чат-Бота

## Быстрая настройка (5 минут)

### Шаг 1: Получите API ключ Gemini (БЕСПЛАТНО)

1. Откройте в браузере: https://makersuite.google.com/app/apikey
2. Войдите через Google аккаунт
3. Нажмите **"Create API Key"**
4. Скопируйте ключ (начинается с `AIza...`)

### Шаг 2: Добавьте ключ в проект

Откройте файл `.env` в корне проекта и найдите строку:

```
GEMINI_API_KEY=
```

Вставьте ваш ключ после знака равно:

```
GEMINI_API_KEY=AIzaSyBj7x2i7FfWqBMH5KtBz9J....
```

Сохраните файл.

### Шаг 3: Очистите кэш

Выполните в терминале:

```bash
php artisan config:clear
```

### ✅ Готово!

Откройте сайт и в правом нижнем углу увидите кнопку чат-бота 🤖

## Как пользоваться

**Примеры вопросов:**

```
"Какие проекты в Туркестанской области?"
"Покажи проблемные вопросы у региона Шымкент"
"Сколько всего проектов?"
"Какие проблемы с высоким приоритетом?"
"Список всех СЭЗ"
"Покажи незаконных недропользователей"
```

Бот умеет:
- ✅ Искать информацию по регионам
- ✅ Показывать данные о проектах
- ✅ Анализировать проблемные вопросы
- ✅ Выдавать статистику по СЭЗ и индустриальным зонам
- ✅ Отвечать на вопросы о недропользователях

## Проблемы?

**Бот не отвечает?**
1. Проверьте, что API ключ добавлен в `.env`
2. Выполните `php artisan config:clear`
3. Проверьте логи: `storage/logs/laravel.log`

**Ошибка в консоли браузера?**
1. Выполните `npm run build`
2. Обновите страницу (Ctrl+F5)
