# 🤖 Чат-Ботты Қалай Пайдалану Керек

## Бір рет серверде баптау қажет

AI-көмекші толық жауаптарды Groq арқылы құрастырады және тек Turkistan Invest жүйесінің рөл бойынша сүзілген деректерін қолданады. Сервер ортасына `GROQ_API_KEY` қосыңыз; кілт браузерге немесе Git репозиторийіне түспеуі тиіс. Groq уақытша қолжетімсіз болса, жергілікті жауап автоматты түрде іске қосылады.

Жүйеге кіргеннен кейін оң жақ төменгі бұрышта чат-бот батырмасын 🤖 көресіз.

## Қалай Пайдалану Керек

**Сұрау мысалдары:**

```
"Түркістан облысында қанша жоба бар?"
"Белсенді мәселелерді көрсет"
"Барлығы қанша жоба бар?"
"Қандай АЭА бар?"
"Недропайдаланушылар тізімі"
"Қандай бөлімдер бар?"
```

Бот не істей алады:
- ✅ Аймақтар бойынша ақпарат іздеу
- ✅ Жобалар туралы мәліметтерді көрсету
- ✅ Мәселелерді талдау
- ✅ АЭА және индустриалды аймақтар бойынша статистика
- ✅ Недропайдаланушылар туралы сұрақтарға жауап беру
- ✅ Навигация және көмек көрсету

## Ақаулықтарды жою

**Бот жауап бермейді?**
1. Дерекқорда деректер бар екенін тексеріңіз
2. Логтарды қараңыз: `storage/logs/laravel.log`

**Браузер консолінде қате?**
1. `npm run build` орындаңыз
2. Бетті жаңартыңыз (Ctrl+F5)
