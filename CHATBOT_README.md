# Turkistan Invest ішкі AI-көмекшісі (Groq + жүйе деректері)

## Сипаттама

AI-көмекші Turkistan Invest жүйесіне ендірілген. Жауапты Groq құрастырады, бірақ оған тек авторизацияланған пайдаланушыға қолжетімді, жүйенің жергілікті дерекқорынан алынған контекст беріледі. Groq қолжетімсіз болса, жергілікті шаблондық жауап автоматты түрде қолданылады.

Пайдаланушылар келесі ақпаратты сұрай алады:
- Аймақтар және инвестициялық көрсеткіштер
- Инвестициялық жобалар
- АЭА (Арнайы экономикалық аймақтар)
- Индустриалды аймақтар
- Пром зоналар
- Жер қойнауын пайдаланушылар
- Мәселелер (Issues)
- Тапсырмалар және олардың күйі

## Орнату

Сервердің қорғалған орта айнымалыларына келесіні қосыңыз:

```env
GROQ_API_KEY=your-secret-key
GROQ_MODEL=llama-3.3-70b-versatile
GROQ_BASE_URL=https://api.groq.com/openai/v1
```

Нақты кілтті Git репозиторийіне немесе frontend кодының ішіне қоспаңыз. Орта айнымалыларын өзгерткеннен кейін Laravel конфигурация кэшін жаңартыңыз.

## Пайдалану

Чат-бот әр беттің оң жақ төменгі бұрышында жүзбелі батырма ретінде көрсетіледі (авторизациядан кейін).

1. **Чатты ашу**: Оң жақ төменгі бұрыштағы робот батырмасын басыңыз
2. **Сұрақ қою**: Енгізу өрісіне сұрақ жазып, Enter немесе жіберу батырмасын басыңыз
3. **Чатты жабу**: Тақырыптағы X батырмасын басыңыз

### Сұрау мысалдары

**Аймақтар туралы:**
- «Барлығы қанша аймақ бар?»
- «Түркістан облысында қандай жобалар бар?»
- «Аймақтар бойынша статистиканы көрсет»

**Жобалар туралы:**
- «Қанша жоба іске асыру сатысында?»
- «Барлығы қанша жоба?»
- «Мәселесі бар жобаларды көрсет»

**Мәселелер туралы:**
- «Белсенді мәселелер қандай?»
- «Барлық шешілмеген мәселелерді көрсет»

**АЭА және индустриалды аймақтар туралы:**
- «Барлық АЭА тізімі»
- «Қандай индустриалды аймақтар бар?»

**Жер қойнауын пайдаланушылар туралы:**
- «Қанша недропайдаланушы бар?»
- «Недропайдаланушылар тізімі»

## Архитектура

### Backend

**Контроллер:** `app/Http/Controllers/ChatController.php`
- `send()` — хабарлама жіберу және жауап алу

**Сервистер:**
- `app/Services/LocalChatService.php` — сұрауды талдау + шаблондық жауап генерациясы
- `app/Services/ChatContextService.php` — дерекқордан мәліметтерді жинау
- `app/Services/GroqService.php` — рөл бойынша сүзілген контекст негізінде толық AI жауабын құрастыру

### Frontend

**Компонент:** `resources/js/components/chat-widget.tsx`
- React компонент, хабарламалар күйін басқару, API-ге сұрау жіберу

### API Endpoint

```
POST   /chat/send      - Хабарлама жіберу
```

## Бот қалай жұмыс істейді

1. **Пайдаланушы сұрақ қояды**
2. **Сұрауды талдау** — жүйе қандай entity туралы сөз болып жатқанын анықтайды (regex арқылы)
3. **Мәліметтерді жинау** — дерекқордан қажетті деректер алынады
4. **Жауапты құрастыру** — шаблондық жауап генераторы қазақ/орыс тілінде жауап береді

## Баптау

### Жаңа деректер түрлерін қосу

1. `LocalChatService.php` ішіндегі `analyzeQuery()` әдісіне regex қосыңыз
2. `ChatContextService.php` ішіне мәліметтерді алу әдісін қосыңыз
3. `LocalChatService.php` ішіне жауапты форматтау әдісін қосыңыз

### Сыртқы түрін өзгерту

`resources/js/components/chat-widget.tsx` компонентін өзгертіңіз.

## Қауіпсіздік

- Барлық сұраулар авторизацияны қажет етеді (`auth` middleware)
- CSRF қорғауы барлық POST сұраулар үшін
- Хабарлама ұзындығы 1000 таңбамен шектелген
- Groq-қа тек ағымдағы сұраққа қажет және пайдаланушы рөлі бойынша сүзілген контекст жіберіледі
- API кілті тек сервер ортасында сақталады және браузерге берілмейді
- Сайтқа қатысы жоқ сұрақтар Groq-қа жіберілмейді

## Ақаулықтарды жою
